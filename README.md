# Book Shop Management System

## 📌 Project Description
Book Shop Management System built using Spring Boot and MySQL.

## 🚀 Features
- Admin Login
- User Login
- Add / Update / Delete Books
- Purchase Books
- Role-based authentication

## 🛠 Tech Stack
- Java
- Spring Boot
- Hibernate (JPA)
- MySQL
- Maven

## ▶ How to Run
1. Clone the repository
2. Configure MySQL in application.properties
3. Run BookShopAppApplication.java
